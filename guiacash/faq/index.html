🚀 Lançamentos e Automação

O Guia Cash possui integração para leitura de dados bancários.

1. Importar Fatura: Use para arquivos CSV/Excel de cartões de crédito. O sistema mapeia as colunas automaticamente.
2. Importar Extrato: Ideal para arquivos de conta corrente (OFX/CSV).
3. Importante: Após importar, o sistema mantém os dados em modo de rascunho até que você valide as categorias.
Este botão executa uma rotina de limpeza e ordenação. Ele remove linhas vazias acidentais, padroniza as fontes para Inter e ordena todos os seus lançamentos por Data (da mais recente para a antiga), garantindo que os dashboards leiam os dados sem erros.
O sistema é visualmente dividido para evitar confusão:
- AZUL (PF): Categorias voltadas a custo de vida (Moradia, Alimentação, Lazer).
- VERDE/TEAL (PJ): Categorias de negócio (Impostos, Custos Diretos, Marketing, RH).
As categorias do PJ alimentam diretamente o DRE na aba Resultados PJ.

📊 Diagnóstico e Indicadores

É o seu termômetro de risco. Ela cruza 6 indicadores principais, como Comprometimento de Caixa e Nível de Alavancagem. - Se o ponteiro estiver no Verde, você tem folga real.
- Se estiver no Vermelho, o sistema indica que sua dívida ou custo fixo ultrapassou sua capacidade de geração de renda.
Ao preencher o credor e a taxa de juros na aba Mapa de Dívidas, o sistema calcula o Custo Efetivo. Ele envia automaticamente o valor total atualizado para a aba Objetivos, criando uma meta de quitação que se atualiza conforme você paga as parcelas.

⚙️ Operacional e Erros

Na aba Resultados PJ. O gráfico de Jornada do Resultado mostra exatamente onde seu dinheiro "escapou" (Impostos -> Custos -> Despesas Operacionais) até chegar no Lucro Líquido. Se houver retiradas para o PF, o sistema mostrará o indicador "Lucro com Interferência PF".
Sim, as colunas TIPO e SUBTIPO são baseadas em validação de dados. Para alterar, você deve acessar a aba de configurações ocultas ou editar o intervalo de validação. Recomenda-se manter o padrão para não quebrar os gráficos do Dashboard.
